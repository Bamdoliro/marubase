== 개요

=== API 서버 경로
[cols="2,5,3"]
|====

=== 응답형식
다음과 같은 공통 응답 형식을 제공합니다.

==== 정상

|====
|데이터가 없을 때|데이터가 하나일 때|데이터가 여러 개일 때

a|[source,json]
----
{}
----

a|[source,json]
----
{
    "code": "OK",
    "message": "ok",
    "data": {}
}
----

a|[source,json]
----
{
    "code": "OK",
    "message": "ok",
    "dataList": []
}
----

|====

==== 에러
응답시 다음과 같은 응답상태 헤더, 응답코드 및 응답메시지를 제공합니다.

|====
|데이터가 없을 때|데이터가 있을 때(Field Error)

a|[source,json]
----
{
    "code": "OK",
    "message": "ok",
    "error": null
}
----

a|[source,json]
----
{
    "code": "OK",
    "message": "ok",
    "error": {
      "name": "message"
    }
}
----

|====
